# 🌲 Triathlon des trois esprits – Visual Novel Ren'Py

Bienvenue dans le **Triathlon des trois esprits**, un conte interactif développé avec **Ren'Py**, mêlant aventure, mystère et magie élémentaire. Ce dépôt contient tout le code source, les assets et les scripts nécessaires pour faire vivre cette histoire dans un format visual novel.

## 📖 Synopsis

Léo, un jeune homme déterminé, rêve de réussir un triathlon légendaire composé de trois épreuves liées aux éléments. Malgré les difficultés, il persévère grâce à l’aide de son amie Mira et aux conseils d’un mystérieux entraîneur. À travers courage et persévérance, il comprend que la vraie victoire est de ne jamais abandonner.

## 🛠️ Technologies utilisées

- [Ren'Py](https://www.renpy.org/) – moteur de visual novel
- Python (intégré à Ren'Py)
- Illustrations originales en style anime
